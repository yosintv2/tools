FAQ – Smart Beautifier
F: Welche Programmiersprachen werden unterstützt?
HTML, CSS und JavaScript werden unterstützt. Der Code-Typ wird automatisch erkannt.
F: Welche Einrückungsart wird verwendet?
Es wird ausschließlich mit Tabulatoren eingerückt. Dieses Tool formatiert den Code einheitlich mit Tabs statt Leerzeichen.
F: Ist es sicher, Code hier einzufügen?
Ja. Alles wird im Browser lokal ausgeführt. Es findet kein Upload statt, und keine Daten werden gespeichert – Ihre Privatsphäre bleibt geschützt.
F: Wie kopiere ich den formatierten Code?
Klicken Sie nach der Formatierung auf „Kopieren“ – der Inhalt wird in die Zwischenablage übernommen.
F: Funktioniert es auch mit fehlerhaftem oder unvollständigem Code?
Ja. Solange der Code soweit syntaktisch erkennbar ist, kann er eingerückt und neu formatiert werden, auch wenn er unvollständig ist.
